# プロジェクト概要

- 言語: TypeScript
- フレームワーク: React (Next.js アプリケーション、App Router)
- UI: MUI v5, MUI X DataGrid
- テスト/ドキュメント: Storybook
- インフラ/API: AWS SDK v3 (S3, Bedrock)
